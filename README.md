# MINARAI CS Chat Front UI

Front UI of text-based chat system for customer support

## Requirements
Node.js (version 5系以上。6.2.2で動作確認済）

## Install && Usage
```
# resolve all dependencies
npm i

# make config file
npm run setup

# fix src/js/conf/development.js
# ex. minarai.url

# build & watch & start server
npm start
```
